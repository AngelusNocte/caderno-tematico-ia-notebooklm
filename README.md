# 📚 Caderno Temático - Inteligência Artificial com NotebookLM

## 🎯 Contexto e Objetivo

Este projeto foi desenvolvido como parte de um desafio da DIO, com foco no uso de Inteligência Artificial como ferramenta de aprendizagem ativa.

O objetivo é utilizar IA para organizar conhecimento, estudar conceitos técnicos e praticar engenharia de prompts.

### 📌 Tema escolhido:
**Inteligência Artificial Generativa aplicada ao desenvolvimento de software**

---

## 🧠 Objetivos de Estudo

- Entender o funcionamento de IAs generativas
- Explorar aplicações práticas em programação
- Aprender engenharia de prompts
- Usar IA como copiloto de aprendizado
- Criar um guia reutilizável de estudo

## 🤖 Engenharia de Prompts

### 🔹 Prompt 1
**Pergunta:**  
O que são IAs generativas?

**Resposta resumida:**  
São modelos de IA capazes de criar conteúdo como texto, imagens e código a partir de dados aprendidos.

---

### 🔹 Prompt 2
**Pergunta:**  
Como IA generativa pode ser usada em desenvolvimento de software?

**Resposta resumida:**  
Pode gerar código automaticamente, sugerir soluções e auxiliar na escrita de APIs e aplicações.

---

### 🔹 Prompt 3
**Pergunta:**  
Explique IA generativa com exemplo prático em Python

**Resposta resumida:**  
Uma IA pode gerar funções Python automaticamente a partir de descrições em linguagem natural.

---

## ⚠️ Dificuldades (Cicatrizes de Prompt)

- Respostas iniciais muito genéricas
- Necessidade de adicionar contexto técnico nos prompts
- Melhores resultados com perguntas específicas e exemplos

## 📘 Miniguia de Estudo

### 📌 Resumo

IAs generativas são modelos capazes de criar novos conteúdos com base em padrões aprendidos em grandes volumes de dados.

---

### 📌 Conceitos principais

- **IA Generativa:** cria conteúdo novo
- **LLM:** modelo de linguagem de grande escala
- **Prompt Engineering:** arte de formular perguntas para IA
- **Copiloto:** IA que auxilia no desenvolvimento

---

### 📌 Glossário

- **Token:** unidade de texto processada pela IA  
- **LLM:** Large Language Model  
- **Prompt:** instrução enviada à IA  
- **Fine-tuning:** ajuste de modelo com dados específicos  

---

### 📌 Prompts reutilizáveis

- Explique [tema] com exemplos práticos  
- Compare [conceito A] e [conceito B]  
- Gere código Python para [função]  
- Resuma o conteúdo em tópicos
